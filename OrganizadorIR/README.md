**Organizador - Imposto de Renda**
===================================

Tem por objetivo ajudar na organização dos documentos para declaração de Imposto de Renda.

Composto de três planilhas em que o usuário informa seus dados pessoais, dados bancários e ganhos no ano.

Possui link entre as planilhas e apenas permite que o usuário digite em células pré-selecionadas.
